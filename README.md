# ⚙️ Major Project: AI Network Fault Prediction System  
### Intelligent Network Monitoring & Fault Detection Tool  

A web-based AI system designed to predict network faults using parameters like CPU usage, latency, packet loss, and bandwidth.  
It uses a **Bayesian Inference model** to analyze network conditions and provide intelligent predictions with suggestions.

---

# 🚀 Project Features  

- 🔍 Network Fault Prediction  
- 🧠 Bayesian Inference Model (pgmpy)  
- 📊 Fault Probability Calculation  
- ⚠️ Risk Level Indicator (Low / Medium / High)  
- 💡 Smart Suggestions  
- 📄 Downloadable Report  
- 📊 Explainable AI (Rule Evaluation)  
- 🌐 Web-Based Interface  

---

# 🧠 Project Working  

Input Parameters → Bayesian Model → Probability → Prediction  

## Input Parameters  

- CPU Usage  
- Latency  
- Packet Loss  
- Bandwidth  

## Output  

- Prediction (Fault / Normal)  
- Probability (%)  
- Risk Level  
- Suggestions  

---

# 📄 Report Generation  

The system generates a downloadable report including:

- Input values  
- Prediction result  
- Probability  
- Risk level  
- Suggestions
